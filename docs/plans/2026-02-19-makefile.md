# Makefile Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a single flat `Makefile` to the project root with build, test, lint, fmt, clean, release, and install targets.

**Architecture:** One `Makefile` at the project root. Variables at the top define binary name, module path, version (from git), LDFLAGS, and target platforms. All targets are `.PHONY`. The `help` target is first so bare `make` prints usage.

**Tech Stack:** GNU Make, Go 1.26, golangci-lint

---

### Task 1: Scaffold the Makefile with variables and `help` target

**Files:**
- Create: `Makefile`

**Step 1: Create the file with variables and `help` target**

```makefile
BINARY  := imgutil
MODULE  := github.com/thisisnotashwin/imgutil
VERSION := $(shell git describe --tags --always --dirty 2>/dev/null || echo "dev")
LDFLAGS := -ldflags "-X $(MODULE)/internal/version.Version=$(VERSION)"
PLATFORMS := linux/amd64 linux/arm64 darwin/amd64 darwin/arm64

.PHONY: help
help: ## Show this help message
	@grep -E '^[a-zA-Z_-]+:.*?## .*$$' $(MAKEFILE_LIST) | \
		awk 'BEGIN {FS = ":.*?## "}; {printf "  \033[36m%-12s\033[0m %s\n", $$1, $$2}'
```

**Step 2: Verify `help` works**

Run: `make help`
Expected: prints a colour-coded target list with `help` shown.

**Step 3: Commit**

```bash
git add Makefile
git commit -m "chore: add Makefile scaffold with help target"
```

### Task 2: Add `build` and `clean` targets

**Files:**
- Modify: `Makefile`

**Step 1: Append targets after `help`**

```makefile
.PHONY: build
build: ## Build binary for current platform → ./bin/imgutil
	@mkdir -p bin
	@go build $(LDFLAGS) -o bin/$(BINARY) ./cmd/imgutil

.PHONY: clean
clean: ## Remove build artefacts (bin/ and dist/)
	@rm -rf bin dist
```

**Step 2: Verify `build` produces a binary**

Run: `make build`
Expected: `./bin/imgutil` exists and `./bin/imgutil --help` prints usage.

Run: `make clean`
Expected: `bin/` directory is removed.

**Step 3: Commit**


### Task 6: Add `release` target for cross-compilation

**Files:**
- Modify: `Makefile`

**Step 1: Append target**

```makefile
.PHONY: release
release: ## Cross-compile for all platforms → ./dist/
	@mkdir -p dist
	@$(foreach PLATFORM,$(PLATFORMS), \
		$(eval OS   := $(word 1,$(subst /, ,$(PLATFORM)))) \
		$(eval ARCH := $(word 2,$(subst /, ,$(PLATFORM)))) \
		echo "Building $(OS)/$(ARCH)..."; \
		GOOS=$(OS) GOARCH=$(ARCH) go build $(LDFLAGS) \
			-o dist/$(BINARY)-$(OS)-$(ARCH) ./cmd/imgutil; \
	)
```

**Step 2: Verify cross-compilation**

Run: `make release`
Expected: `dist/` contains four binaries:
```
dist/imgutil-linux-amd64
dist/imgutil-linux-arm64
dist/imgutil-darwin-amd64
dist/imgutil-darwin-arm64
```

**Step 3: Commit**

```bash
git add Makefile
git commit -m "chore: add release target with cross-compilation to Makefile"
```
